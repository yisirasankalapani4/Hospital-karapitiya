Available Laboratory Tests

Comprehensive diagnostic testing services available 24/7 for all maternal and infant healthcare needs.

Prenatal Testing

  • Complete Blood Count (CBC)
  • Blood glucose screening
  • Thyroid function tests
  • VDRL/RPR screening
  • HIV screening
  • Hepatitis B screening

Pregnancy Monitoring

  • HCG levels
  • AFP screening
  • Genetic screening tests
  • Urinalysis
  • Protein levels
  • Blood pressure monitoring

Newborn Tests

  • Blood type and Rh factor
  • Bilirubin levels
  • Metabolic screening
  • Hearing screening
  • Genetic disorder screening
  • Infection screening
